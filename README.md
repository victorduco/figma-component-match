# Figma Component Match

A playground Figma plugin built for the workshop on May 14, 2026.

## Features

- **Run** — scan the "Components" page and list all components
- **Find** — highlight selected components across the current page
- **Replace** — swap matched frames with real component instances
- **Wrap in Template Section** — wrap selected nodes into a copy of the "Template Section" frame, centered with padding

## Setup

1. Open Figma → Plugins → Development → Import plugin from manifest
2. Select `manifest.json` from this repo
3. Run the plugin from the Plugins menu
